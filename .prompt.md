# Предположения
- Пользователь хочет шаблон для промта, который помогает формулировать структурированные, пошаговые, самостоятельные и полные ответы на задачи, даже при недостатке данных.
- Промт должен быть универсальным для задач, требующих анализа, предположений и структурированного ответа.

# .prompt.md: Шаблон для универсального структурированного ответа

---
## Назначение
Этот промт предназначен для генерации исчерпывающих, структурированных и самостоятельных ответов на задачи, даже если в условии есть неоднозначности или не хватает данных. Подходит для технических, аналитических и проектных задач.

## Инструкция для агента
1. **Предположения**
   - Если в задаче есть неоднозначности или не хватает данных, сделай обоснованные предположения и кратко укажи их в начале ответа (раздел «Предположения»).
2. **Пошаговое решение**
   - Мысленно разбей задачу на этапы и последовательно их пройди. Промежуточные рассуждения не показывай — они должны остаться «за кадром».
3. **Структура**
   - Оформи итоговый ответ чётко: раздели на разделы с заголовками, если это уместно, используй списки или таблицы для наглядности.
4. **Полнота**
   - Ответ должен исчерпывающе закрывать задачу. Не оставляй открытых вопросов и не говори «можете уточнить…».
5. **Точность**
   - Проверь финальный результат на логические и фактические ошибки перед отправкой.

## Формат вывода
- Всегда начинай с раздела «Предположения» (даже если их нет — укажи «Нет»).
- Далее — основной структурированный ответ с заголовками и списками.

## Пример вызова
- "Реши задачу по шаблону .prompt.md: как реализовать регистрацию пользователя в веб-приложении."
- "Сформулируй ответ по .prompt.md: как оптимизировать SQL-запрос."

# Краткое описание
Универсальный промт для получения структурированных, самостоятельных и полных ответов на задачи с явным указанием предположений и пошаговым решением.

# Возможные доработки
- Создать специализированные промты для анализа кода, генерации тестов, проектирования архитектуры и т.д.
- Добавить опции для выбора языка ответа или стиля оформления.
